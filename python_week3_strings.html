Character vs String in Python

In Python, the difference between character and string is a little different compared to languages like C or Java.

Character in Python

Python does not have a separate char data type. A single character is treated as a string with length = 1.

Example:

ch = 'A'

print(type(ch))
print(len(ch))

Output:

<class 'str'>
1
Here 'A' is not a character type. In Python it is a string of length 1.

1. Introduction to Strings

In Python, a string is a sequence of characters enclosed within quotes.

  • Single Quotes : 'Hello'
  • Double Quotes : "Hello"
  • Triple Quotes : '''Hello'''
s1 = 'Hello'
s2 = "Python"
s3 = '''Welcome to Python Programming'''
Strings are immutable which means their value cannot be changed after creation.

3. String Slicing

String Slicing in Python

Slicing extracts a portion of a string using the syntax:

string[start:end]

Example Program

text = "Python Programming"

# Extract characters from index 0 to 5
print(text[0:6])

# Extract characters from index 7 to the end
print(text[7:])

Output

Python
Programming

Explanation:

text[0:6] → characters from index 0 to 5 (end index is not included)
text[7:] → characters from index 7 to the end of the string

Using Step in Slicing

The step value allows skipping characters.

text = "PYTHON"

# Skip every 2 characters
print(text[::2])

# Start from index 1 and skip every 2 characters
print(text[1::2])

# Reverse the string using negative step
print(text[::-1])

# Start from index 0 to 6 and skip every 3 characters
print(text[0:6:3])

Output

PTO
YHN
NOHTYP
PH

Explanation

Code Meaning Result
text[::2] Start from beginning, skip every 2 characters PTO
text[1::2] Start from index 1 and skip every 2 characters YHN
text[::-1] Reverse the string NOHTYP
text[0:6:3] From index 0 to 6, skip every 3 characters PH

Note for Students

Python slicing format:

string[start : end : step]

start → starting index
end → stopping index (not included)
step → number of characters to skip

6. Laboratory Programs – String Manipulation

Program 1 – Strip Characters from String

txt = input()
char_to_strip = input()

cleaned_txt = txt.strip(char_to_strip)

print("Cleaned text : ", cleaned_txt)
Function Used : strip() removes specified characters from beginning and end of the string.

Program 2 – Find Length and Copy String

# Read the input string
A = input().strip()

# Find the length using len()
length = len(A)
print("Length of the string:", length)

# Copy the content of A to B
B = A

# Print the copied string
print("Copied string:", B, end="")
Functions Used : len(), strip()

Program 3 – Generate Email Address

first_name = input()
last_name = input()
role = input()
company = input()

first_name = first_name.lower()
role = role.lower()
company = company.lower()

if role == "admin":
    suffix = "@admin." + company + ".com"
else:
    suffix = "@" + company + ".com"

email = first_name[0] + last_name.lower() + suffix

print(email)
Functions Used : lower(), indexing, string concatenation

Program 4 – Reverse Second String and Concatenate

string1 = input()
string2 = input()

reversed_string2 = string2[::-1]

result = string1 + reversed_string2

print(result)
Concept Used : String slicing for reversing a string.

Program 5 – Replace Character in Two Strings

string1 = input()
string2 = input()

char_to_replace = input()
char_to_replace_with = input()

str1 = ""
for char in string1:
    if char == char_to_replace:
        str1 += char_to_replace_with
    else:
        str1 += char

str2 = ""
for char in string2:
    if char == char_to_replace:
        str2 += char_to_replace_with
    else:
        str2 += char

result = "{} {}".format(str1, str2)

print(result)
Concept Used : for loop, character comparison, string building, format()
