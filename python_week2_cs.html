Week 2: Control Structures in Python

Introduction

“Till now, our program executes line by line. But what if we want decisions? Or repetition?”

👉 Control structures control the flow of execution of a program.

Normally Python executes:

Line 1 Line 2 Line 3

But control structures allow:

  • ✔️ Decision making
  • ✔️ Repeating tasks
  • ✔️ Skipping parts of code

Types of Control Structures in Python

  1. Decision Making Statements
    • if
    • if-else
    • if-elif-else
  2. Looping Statements
    • for
    • while
  3. Jump Statements
    • break
    • continue
    • pass

PART 1: Decision Making Statements

1️⃣ if Statement

Syntax:

if condition: statement

Example:

age = 18 if age >= 18: print("You are eligible to vote")

Explain clearly:

  • Condition must return True or False
  • Indentation is mandatory in Python

Multiple Condition Example:

if x > y and x > z: print(f"The maximum number is: {x}")

Real Example:

day = int(input()) doctor = "" if day == 1 or day == 8 or day == 14: doctor = "Dr. Smith"

2️⃣ if-else Statement

num = 10 if num % 2 == 0: print("Even number") else: print("Odd number")
  • Only one block executes
  • Else runs when condition is False

3️⃣ if-elif-else Ladder

marks = 75 if marks >= 90: print("Grade A") elif marks >= 75: print("Grade B") elif marks >= 50: print("Grade C") else: print("Fail")
  • Multiple conditions
  • Checked top to bottom
  • First True condition executes

PART 2: Looping Statements

What if I want to print Hello 100 times?

Instead of writing print 100 times — use loops.

1️⃣ for Loop

Used when number of repetitions is known.

for i in range(5): print("Hello")

Explain:

  • range(5) → 0 to 4
  • Loop runs 5 times

Another Example:

for i in range(1, 6): print(i)
  • Start = 1
  • End = 6 (excluded)
  • Output: 1 2 3 4 5

2️⃣ while Loop

Used when condition is based.

count = 1 while count <= 5: print(count) count += 1
  • Loop runs while condition is True
  • Important: Update variable
  • Otherwise → Infinite loop

Difference: while vs do-while

while loop do-while loop
Condition checked first Condition checked last
May not execute even once Executes at least once

Note: Python does NOT have a do-while loop. Instead, we simulate it using while True.

x = 5 while True: print(x) x += 1 if x >= 5: break

PART 3: Jump Statements

break

Stops loop immediately.

for i in range(1, 10): if i == 5: break print(i)

continue

Skips current iteration.

for i in range(1, 6): if i == 3: continue print(i)

Output: 1 2 4 5
