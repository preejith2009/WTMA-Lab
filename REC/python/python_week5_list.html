Week 4 - Python Non-Primitive Data Structures

Python provides built-in data structures to store collections of data efficiently. The four major non-primitive data structures are:

  • List – Ordered and mutable
  • Tuple – Ordered and immutable
  • Set – Unordered and unique elements
  • Dictionary – Key-value pair structure

Difference Between List, Tuple, Set and Dictionary

List        Tuple       Set         Dictionary
[]          ()          {}          {key:value}

Ordered     Ordered     Unordered   Ordered
Mutable     Immutable   Mutable     Mutable

Duplicates  Allowed     Not Allowed Keys Not Allowed

Lists and tuples store ordered data, but tuples cannot be modified once created. Sets remove duplicates automatically, and dictionaries store data as key-value pairs for fast lookup.

Python List

A List is a dynamic and flexible data structure used to store multiple values. It supports different data types and allows modification of elements.

Example

a = [23, 45, 12, 67, 34]
a.append(89)
a.remove(45)
print(a)
Output: [23, 12, 67, 34, 89]

Applications of List

  • Store and manipulate data easily
  • Used in stacks (LIFO) and queues (FIFO)
  • Perform iteration and calculations
  • Dynamic data storage (grows/shrinks)
  • String processing and tokenization

Program 1 – Arrange Negative Numbers Before Positive Numbers

Program
input_str = input()
numbers = [int(num) for num in input_str.strip('[]').split(',')]

negatives = []
positives = []

for num in numbers:
    if num < 0:
        negatives.append(num)
    else:
        positives.append(num)

arranged_numbers = negatives + positives
print("List = ", arranged_numbers)
Explanation
  • The program reads a list of numbers as input.
  • strip('[]') removes square brackets from the input.
  • split(',') separates numbers using commas.
  • The list comprehension converts each value into an integer.
  • Two empty lists are created:
    • negatives – stores negative numbers
    • positives – stores positive numbers
  • The loop checks each number in the list.
  • If the number is negative it is stored in the negatives list.
  • Otherwise it is stored in the positives list.
  • Both lists are combined so that negative numbers appear first.
Example Input : [3,-2,5,-7,1]
Output : [-2,-7,3,5,1]

Program 2 – Rotate List Elements by 2 Positions

Program
n = int(input())
workers = input().split()

rotated = workers[-2:] + workers[:-2]

print(' '.join(rotated))
Explanation
  • The program reads the number of elements n.
  • The next input reads worker names separated by spaces.
  • split() converts the input into a list.
  • The list is rotated two positions to the right using slicing.
  • workers[-2:] returns the last two elements.
  • workers[:-2] returns the remaining elements.
  • The two lists are combined to produce the rotated list.
  • join() prints the list as a space separated string.
Input : A B C D E
Output : D E A B C

Program 3 – Find the Smallest Missing Positive Number

Program
n = int(input())
st = input().split()
a = [int(num) for num in st]

positive_numbers = []

for num in a:
    if num > 0 and num not in positive_numbers:
        positive_numbers.append(num)

index = 1
while True:
    if index not in positive_numbers:
        print(index)
        break
    index += 1
Explanation
  • The program reads a list of integers.
  • The list comprehension converts input strings into integers.
  • A new list positive_numbers stores only positive numbers.
  • Duplicates are avoided using the condition num not in positive_numbers.
  • The program then checks numbers starting from 1.
  • The first number not found in the list is printed.
Input : 3 4 -1 1 2
Output : 5

Program 4 – Find the Largest Divisor from the List

Program
n = int(input())
st = input().split(',')

lst = [int(n) for n in st]

out = []

for ind in range(n):

    divlst = [-1]

    for div in range(n):

        if lst[ind] % lst[div] == 0 and ind != div:
            divlst.append(lst[div])

    out.append(max(divlst))

print(out)
Explanation
  • The program reads numbers separated by commas.
  • List comprehension converts them into integers.
  • For each number, the program checks which numbers divide it.
  • The condition used is lst[ind] % lst[div] == 0.
  • All divisors are stored in the list divlst.
  • max(divlst) finds the largest divisor.
  • The result for each number is stored in the list out.
Input : 2,3,4,8
Output : [-1,-1,2,4]

Program 5 – Replace List Elements Using Conditions

Program
n=int(input())
l=[]

for i in range(n):
    l.append(int(input()))

l.sort()
print("Original List:",l)

for i in range(len(l)):
    if i==0:
        l[i]=0
    elif i%2==0:
        l[i]=l[i]*l[i]*l[i]
    else:
        l[i]=l[i]*l[i]

print("Replaced List:",l)
Explanation
  • The program reads n numbers from the user.
  • All numbers are stored in a list.
  • The list is sorted using sort().
  • The program modifies elements based on index position.
Rules Applied
  • Index 0 → replaced with 0
  • Even index → square of the number
  • Odd index → cube of the number
Original List : [1,2,3,4,5]
Replaced List : [0,4,27,16,125]

Python Tuple

A Tuple is similar to a list but immutable (cannot be changed after creation). It is useful when data should remain constant.

Example

p = (10, 20)
print(p)
Output: (10, 20)

Applications of Tuple

  • Store fixed data like coordinates
  • Used as dictionary keys
  • Store heterogeneous data

Python Set

A Set is an unordered collection of unique elements. It automatically removes duplicates and supports mathematical operations.

Example

a = [1, 2, 2, 3, 4, 4]
u = set(a)
print(u)
Output: {1, 2, 3, 4}

Applications of Set

  • Removing duplicate elements
  • Performing union, intersection, difference
  • Fast membership checking
  • Finding common elements between datasets

Python Dictionary

A Dictionary stores data in key-value pairs and allows fast access using keys. Each key must be unique.

Example

rec = {
 'id': 1,
 'name': 'John',
 'age': 25
}
print(rec['name'])
Output: John

Grouping Data by Keys

Dictionaries can be used to group data by certain keys. This is very useful in data analysis and aggregation tasks.

Example
emp = [
    {'name': 'prajj', 'dept': 'HR'},
    {'name': 'brij', 'dept': 'IT'},
    {'name': 'kareena', 'dept': 'HR'},
    {'name': 'aryan', 'dept': 'IT'}
]

grp = {}

for e in emp:
    dept = e['dept']
    grp.setdefault(dept, []).append(e['name'])

print(grp)
Output: {'HR': ['prajj', 'kareena'], 'IT': ['brij', 'aryan']}
Explanation
  • The program starts with a list emp, where each element is a dictionary containing employee name and department.
  • An empty dictionary grp is created to store grouped data.
  • The loop for e in emp: iterates through each employee record.
  • For each employee, the department is accessed using: e['dept'].
  • setdefault(dept, []) checks whether the department already exists as a key in the dictionary:
    • If it exists → returns the existing list
    • If not → creates a new empty list for that department
  • .append(e['name']) adds the employee name to the corresponding department list.
  • This process repeats for all employees, grouping names based on department.
  • Finally, the grouped dictionary is printed.

Applications of Dictionary

  • Database record representation
  • Counting frequency of elements
  • Fast lookup tables
  • Handling JSON data
  • Grouping data by keys

When to Use Which Data Structure?

  • Use List → when order matters and data changes frequently
  • Use Tuple → when data should not change
  • Use Set → when uniqueness is required
  • Use Dictionary → when data is in key-value form
