Week 4 - Python Non-Primitive Data Structures - LIST

Python provides several built-in data structures that allow us to store collections of data. These are called Non-Primitive Data Structures because they can hold multiple values instead of a single value.

The four built-in non-primitive data structures in Python are:

  • List – Ordered collection of elements that can be changed.
  • Tuple – Ordered collection but cannot be modified.
  • Set – Unordered collection of unique elements.
  • Dictionary – Collection of key-value pairs.

In our syllabus we mainly focus on the List data structure.

Python List

A List is one of the most commonly used data structures in Python. It is used to store multiple items in a single variable.

Characteristics of Lists

  • Lists are ordered.
  • Lists are mutable (can be modified).
  • Lists allow duplicate values.
  • Lists can contain different data types.

Creating a List

numbers = [10, 20, 30, 40]
names = ["Alice", "Bob", "Charlie"]

Accessing List Elements

numbers = [10,20,30]

print(numbers[0])   # first element
print(numbers[1])   # second element

Common List Operations

  • append() – Adds an element at the end
  • insert() – Adds an element at a specific position
  • remove() – Removes an element
  • pop() – Removes element using index
  • len() – Returns length of list

Program 1 – Arrange Negative Numbers Before Positive Numbers

Program
input_str = input()
numbers = [int(num) for num in input_str.strip('[]').split(',')]

negatives = []
positives = []

for num in numbers:
    if num < 0:
        negatives.append(num)
    else:
        positives.append(num)

arranged_numbers = negatives + positives
print("List = ", arranged_numbers)
Explanation
  • The program reads a list of numbers as input.
  • strip('[]') removes square brackets from the input.
  • split(',') separates numbers using commas.
  • The list comprehension converts each value into an integer.
  • Two empty lists are created:
    • negatives – stores negative numbers
    • positives – stores positive numbers
  • The loop checks each number in the list.
  • If the number is negative it is stored in the negatives list.
  • Otherwise it is stored in the positives list.
  • Both lists are combined so that negative numbers appear first.
Example Input : [3,-2,5,-7,1]
Output : [-2,-7,3,5,1]

Program 2 – Rotate List Elements by 2 Positions

Program
n = int(input())
workers = input().split()

rotated = workers[-2:] + workers[:-2]

print(' '.join(rotated))
Explanation
  • The program reads the number of elements n.
  • The next input reads worker names separated by spaces.
  • split() converts the input into a list.
  • The list is rotated two positions to the right using slicing.
  • workers[-2:] returns the last two elements.
  • workers[:-2] returns the remaining elements.
  • The two lists are combined to produce the rotated list.
  • join() prints the list as a space separated string.
Input : A B C D E
Output : D E A B C

Program 3 – Find the Smallest Missing Positive Number

Program
n = int(input())
st = input().split()
a = [int(num) for num in st]

positive_numbers = []

for num in a:
    if num > 0 and num not in positive_numbers:
        positive_numbers.append(num)

index = 1
while True:
    if index not in positive_numbers:
        print(index)
        break
    index += 1
Explanation
  • The program reads a list of integers.
  • The list comprehension converts input strings into integers.
  • A new list positive_numbers stores only positive numbers.
  • Duplicates are avoided using the condition num not in positive_numbers.
  • The program then checks numbers starting from 1.
  • The first number not found in the list is printed.
Input : 3 4 -1 1 2
Output : 5

Program 4 – Find the Largest Divisor from the List

Program
n = int(input())
st = input().split(',')

lst = [int(n) for n in st]

out = []

for ind in range(n):

    divlst = [-1]

    for div in range(n):

        if lst[ind] % lst[div] == 0 and ind != div:
            divlst.append(lst[div])

    out.append(max(divlst))

print(out)
Explanation
  • The program reads numbers separated by commas.
  • List comprehension converts them into integers.
  • For each number, the program checks which numbers divide it.
  • The condition used is lst[ind] % lst[div] == 0.
  • All divisors are stored in the list divlst.
  • max(divlst) finds the largest divisor.
  • The result for each number is stored in the list out.
Input : 2,3,4,8
Output : [-1,-1,2,4]

Program 5 – Replace List Elements Using Conditions

Program
n=int(input())
l=[]

for i in range(n):
    l.append(int(input()))

l.sort()
print("Original List:",l)

for i in range(len(l)):
    if i==0:
        l[i]=0
    elif i%2==0:
        l[i]=l[i]*l[i]*l[i]
    else:
        l[i]=l[i]*l[i]

print("Replaced List:",l)
Explanation
  • The program reads n numbers from the user.
  • All numbers are stored in a list.
  • The list is sorted using sort().
  • The program modifies elements based on index position.
Rules Applied
  • Index 0 → replaced with 0
  • Even index → square of the number
  • Odd index → cube of the number
Original List : [1,2,3,4,5]
Replaced List : [0,4,27,16,125]
