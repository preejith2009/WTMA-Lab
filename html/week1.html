Lab Programmes Hands-on Practice
  1. Design a simple static HTML web page using:
    • Ordered lists
    • Unordered lists
    • Tables

    Display and maintain student record information in a structured format.

  2. Design static web pages for an online book store website:
    • Home Page: Static home page with three frames
    • Login Page
    • Catalogue Page: Display all available books in a table
    • Registration Page
  3. Design an HTML web page using anchor (<a>) tags and an iframe to display linked content within the same page.

    The iframe should load a web page containing an image using the <img> tag.

  4. Create a web page using HTML that contains a form to collect student information including:
    • Name
    • Email
    • Branch
    • Gender
    • Address
  5. Create a web page to embed a map along with:
    • Hot spots
    • Frames
    • Links
