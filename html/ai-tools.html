AI Tools for WTMA Practicals

Introduction to AI-assisted tools for Web Technology & Mobile Applications

Visual Studio Code

A powerful code editor used for writing, editing, and debugging HTML, CSS, JavaScript, and other programming languages. Supports extensions for AI assistance, version control, and efficient workflow.

Code Editor
Workik HTML Code Generator

Workik HTML Code Generator allows you to visually create web pages and instantly get the HTML code. It simplifies building responsive layouts and components without manually writing all the code.

UI & Code Generator
CodePen

CodePen is an online code editor and playground that allows you to write HTML, CSS, and JavaScript and see the output instantly in one screen. Ideal for prototyping, experimenting, and sharing web development projects.

Online Editor
GitHub

GitHub is a platform for hosting and collaborating on code projects. It is widely used for creating and deploying static websites using GitHub Pages, version control, and collaboration with others.

Version Control & Hosting
Bootstrap

Bootstrap is a free, open-source front-end framework used for building responsive and mobile-first websites quickly and easily. It was originally developed by Twitter.

Front-End Framework
Google AI Studio

Google AI Studio is a cloud-based platform that allows developers and students to experiment with AI models, run machine learning workflows, and build AI-powered applications efficiently.

AI Development Tool
Academic Usage Guidelines
  • AI tools should be used for learning and assistance only.
  • Students must understand and explain generated code.
  • Blind copy-paste of AI output is not permitted.
  • Faculty may ask viva questions on submitted programs.
