WEB TECHNOLOGY & MOBILE APPLICATION

Subject Code: AI23431 | PC | L-T-P-C: 1-0-4-3

Artificial Intelligence & Data Science and Artificial Intelligence & Machine Learning

Course Objectives

  • Provide foundational knowledge and practical skills in creating and structuring web pages using HTML.
  • Understand and practice Embedded Dynamic Scripting on Client-side Internet Programming.
  • Implement Server Side Scripting.
  • Understand Android Application Design.
  • Gain a basic understanding of Android application development.

Course Units

UNIT I: Web Site Basics, HTML 5.2, CSS 3.0

Internet Protocols â€“ HTTP, Understanding Internet, Difference between website and application server, Internet technology overview, HTML5.2 and CSS 3.0

UNIT II: JavaScript

Introduction to Scripting, Core features, Data types and Variables, Operators, Expressions, Functions, Arrays, Objects, DOM, Event Handling, JSON, Introduction to AJAX.

UNIT III: Server Side Programming

Servlets: Java Servlet Architecture, Servlet Life Cycle, Form GET and POST actions, Session Handling, Cookies. DATABASE CONNECTIVITY: JDBC perspectives and examples. JSP: Java Server Pages, JSTL, Creating HTML forms embedding JSP code.

UNIT IV: Mobile Application Development

Android Application Design Essentials: Anatomy of Android applications, Android terminologies, Application Context, Activities, Services, Intents, Receiving and Broadcasting Intents, Manifest File, Permissions.

UNIT V: Android User Interface Design

User Interface elements, Designing UI with Layouts, Drawing and Animation, Testing Android applications, Publishing Android applications.

Total Contact Hours: 15

Course Outcomes

  • Design and develop basic web pages with proper structure and semantic elements, ensuring accessibility and functionality.
  • Design and implement dynamic web pages with validation using JavaScript objects and event handling mechanisms.
  • Design and implement simple webpages to learn JSP and Servlets.
  • Design and implement simple Android application designs.
  • Identify various concepts of mobile programming that make it unique from programming for other platforms.
