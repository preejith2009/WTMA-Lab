Mobile Application Programming

Android Development using Android Studio

1️⃣ Introduction to Android

Android is an open-source mobile operating system developed by Google. It is widely used to develop applications for smartphones, tablets and wearable devices.

  • Uses Java / Kotlin programming
  • UI designed using XML
  • Developed using Android Studio IDE
2️⃣ Android UI Screen Elements
UI Element Description
TextView Displays text
EditText User input field
Button Triggers action
ImageView Displays images
CheckBox Multiple selections
RadioButton Single selection
Spinner Dropdown list
3️⃣ Designing User Interfaces with Layouts

Layouts arrange UI elements on the screen.

  • LinearLayout
  • RelativeLayout
  • ConstraintLayout
  • FrameLayout
  • TableLayout

Example:


<LinearLayout
android:orientation="vertical"
android:layout_width="match_parent"
android:layout_height="match_parent">

<TextView android:text="Username"/>

<EditText android:id="@+id/txtUser"/>

<Button android:text="Login"/>

</LinearLayout>

1️⃣1️⃣ Android Manifest File

The AndroidManifest.xml file contains essential information about the application.


<manifest package="com.example.app">

<application>

<activity android:name=".MainActivity">

<intent-filter>
<action android:name="android.intent.action.MAIN"/>
<category android:name="android.intent.category.LAUNCHER"/>
</intent-filter>

</activity>

</application>

</manifest>
1️⃣2️⃣ Android Intents

Intents are used for communication between Android components.


Intent intent = new Intent(MainActivity.this, SecondActivity.class);
startActivity(intent);
1️⃣3️⃣ Android Permissions

Permissions allow apps to access sensitive device features.


<uses-permission android:name="android.permission.INTERNET"/>

<uses-permission android:name="android.permission.CAMERA"/>
