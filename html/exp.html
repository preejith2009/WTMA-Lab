Web Development Experiments

Client Side
HTML Programmes
Programme 1 – Design a simple static HTML web page using ordered lists, unordered lists, and tables to display and maintain student record information in a structured format.
Programme 2 – Design the following static web pages required for an online book store web site.
a) HOME PAGE: The static home page must contain three frames.
b) LOGIN PAGE
c) CATOLOGUE PAGE: The catalogue page should contain the details of all the books available in the web site in a table.
d) REGISTRATION PAGE
Programme 3 – create a web page using HTML that contains a form to collect student information including name, email, branch, gender, and address.
CSS Programmes
Programme 1 – Develop and demonstrate the usage of inline, internal and external style sheet using CSS.
Programme 2 – Design a Student Record Web Page Using CSS Keyframe Animation.
JavaScript Programmes
Programme 1 – Write JavaScript to validate the following fields of the Registration page.
a) First Name (Name should contains alphabets and the length should not be less than 6 characters).
b) Password (Password should not be less than 6 characters length).
c) E-mail id (should not contain any invalid and must follow the standard pattern name@domain.com)
d) Mobile Number (Phone number should contain 10 digits only).
e) Last Name and Address (should not be Empty).
Programme 2 – Write a Javscript code to display a Pop Up "Form Submitted" when User clicks on submit Button on above Form.
Programme 3 – Write an HTML page including any required JavaScript that takes a number from text field in the range of 0 to 999 and shows it in words. It should not accept four and above digits, alphabets and special characters.
AJAX Programmes
Programme 1 – Create a simple web page that fetches student data from a text file using AJAX and displays it dynamically without reloading the page.
JSON Programmes
Programme 2 – Create a simple web page that fetches student records in JSON format and displays them dynamically using JavaScript.
Server Side
Servlet Programmes
Programme 1 – Write a Servlet program that prints "Hello, World!" when accessed through a browser.
Programme 2 – Create a web form that accepts a user's name and age. Write a Servlet to process the form data and display it back on the browser.
Programme 3 – Write a Servlet to demonstrate session tracking using HttpSession. Implement a simple login system where the user's session is tracked.
JSP Programmes
Programme 1 – create a JSP web page that accepts user input via HTML form and displays the submitted data using both GET and POST methods.
Mobile App Development Programmes
Programmes using Android Studio
Programme 1 – Develop an Android application using controls like Button, TextView, EditText for designing a calculator having basic functionality like Addition, Subtraction, multiplication, and Division.
Programme 2 – Develop an application to change the font and color of the text and display toast message when the user presses the button.
Programme 3 – Develop a mobile application to send an email.
