Week 5 - Python Functions

A function is a block of reusable code used to perform a specific task. Functions help in reducing code repetition and improving readability.

Key Concepts of Functions

  • Functions are defined using def keyword
  • They can take arguments
  • They can return values using return
  • Support for default arguments and multiple arguments

Understanding Functions in Detail

Basic Function Syntax
def function_name(parameters):
    # Step 1: Input processing
    # Step 2: Logic / computation
    return value
Explanation of Each Block
  • def → Keyword used to define a function
  • function_name → Name of the function (user-defined)
  • parameters → Inputs passed to the function
  • : → Indicates start of function block
  • Indented block → Contains logic of the function
  • return → Sends result back to the caller

Function Arguments

Arguments are values passed to a function when it is called.

Example
def add(a, b):
    return a + b

print(add(5, 3))
  • a, b → Parameters
  • 5, 3 → Arguments passed during function call

Types of Arguments

  • Positional Arguments → Values passed in order
  • Default Arguments → Predefined values
  • *args → Multiple values passed as tuple
Default Argument Example
def greet(name="Student"):
    print("Hello", name)

greet()
greet("John")

How Arguments are Passed

  • Values are passed from function call → to parameters
  • Execution happens inside function
  • Result is returned back
Flow Representation
Function Call → Parameters → Execution → Return → Output

Return Statement

The return statement is used to send a result back to the function call.

Example
def square(n):
    return n * n

result = square(4)
print(result)
  • Function computes value
  • return sends result back
  • Value can be stored in a variable

Important Points

  • A function can return multiple values
  • If no return → function returns None
  • Execution stops after return
Multiple Return Example
def calc(a, b):
    return a+b, a*b

sum_val, prod_val = calc(2,3)
💡 Tip for Students:
Always use functions to break large problems into smaller parts. This makes debugging and understanding easier.

Python Function Programs Explanation

Program 1 – Salary Calculation using Functions

Program
def calculate_gross_salary(hours_worked, hourly_rate=20.0):
    return hours_worked * hourly_rate

def calculate_deductions(gross_salary, deduction_rate=0.15):
    return deduction_rate * gross_salary

def calculate_net_salary(gross_salary, deductions):
    return gross_salary - deductions

def print_salary_details(hours_worked, hourly_rate, monthly_salary):
    print(f"Hours Worked: {hours_worked:.2f} hours")
    print(f"Hourly Rate: {hourly_rate:.2f} per hour")
    print(f"Net Salary: {monthly_salary:.2f}")

hours_worked = float(input())
hourly_rate = float(input())

gross_salary = calculate_gross_salary(hours_worked, hourly_rate)
deductions = calculate_deductions(gross_salary)
monthly_salary = calculate_net_salary(gross_salary, deductions)

print_salary_details(hours_worked, hourly_rate, monthly_salary)
Explanation
  • Functions are used to divide the salary calculation into smaller tasks.
  • Gross salary is calculated using hours and rate.
  • Deductions are calculated as 15% of gross salary.
  • Net salary is calculated by subtracting deductions.
Input: 40 20
Output: Net Salary displayed

Program 2 – Calculate Hypotenuse

Program
def calculate_hypotenuse(a, b):
    return (pow(a,2) + pow(b,2))**0.5

side1 = float(input())
side2 = float(input())

result = calculate_hypotenuse(side1, side2)
print(f"The length of the hypotenuse is: {result:.2f}")
Explanation
  • Uses Pythagorean theorem.
  • Function returns square root of sum of squares.
Input: 3 4
Output: 5.00

Program 3 – Print Odd Numbers using *args

Program
def find_odd_numbers(*args):
    first = True
    for n in args:
        for i in range(1, n + 1, 2):
            if not first:
                print(", ", end="")
            print(i, end="")
            first = False

user_input = input()
numbers = map(int, user_input.split())

find_odd_numbers(*numbers)
Explanation
  • *args allows multiple inputs.
  • Loop generates odd numbers.
  • Comma formatting handled manually.
Input: 5 3
Output: 1,3,5,1,3

Program 4 – Find Maximum of Three Numbers

Program
def find_max(a, b, c):
    max_val = a
    if b > max_val:
        max_val = b
    if c > max_val:
        max_val = c
    return max_val

num1, num2, num3 = map(float, input().split())

maximum = find_max(num1, num2, num3)
print(f"{maximum:.2f}")
Explanation
  • Compares three numbers using conditional statements.
  • Returns the largest value.
Input: 5 8 3
Output: 8.00

Program 5 – Count Even and Odd Numbers

Program
n = int(input())
array_nums = [int(i) for i in input().split()[:n]]

odd_ctr = len(list(filter(lambda x: (x % 2 != 0), array_nums)))
even_ctr = len(list(filter(lambda x: (x % 2 == 0), array_nums)))

print(even_ctr)
print(odd_ctr)
Explanation
  • Uses lambda and filter functions.
  • Counts even and odd numbers separately.
Input: 5
1 2 3 4 5
Output: 2 3
